# Running locally

All commands are run from the repository root.

## Development servers

**Web app (SolidStart):**

```bash
pnpm dev
```

Opens at [http://localhost:5173](http://localhost:5173).

**Docs site (Fumadocs/Next.js):**

```bash
pnpm dev:docs
```

Opens at [http://localhost:3000](http://localhost:3000). Documentation lives at [http://localhost:3000/docs](http://localhost:3000/docs).

Run each in a separate terminal if you want both at once.

## Build for production

- **Build everything:** `pnpm build`
- **Web only:** `pnpm build:web`
- **Docs only:** `pnpm build:docs`

## Start production builds

- **Web** (after `pnpm build:web`): `pnpm --filter adapterhub-web start`
- **Docs** (after `pnpm build:docs`): `pnpm --filter adapterhub-docs start`

See [Repo architecture](/docs/repo-architecture) for how the monorepo is structured.
